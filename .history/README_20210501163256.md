This projects featured by api integration with web application. In this projects I used [The sports db api](https://www.thesportsdb.com/api.php). Its allows applications to communicate with backend systems. The sports db api is an sport news portal. By using this api you can easily build an e-sport news website. Its easy to apply your application. This api provides all kind of sports information and sport news also provides player information.

### What is an API integration?

An API integration is the connection between two or more applications, via their APIs, that lets those systems exchange data. API integrations power processes throughout many high-performing businesses that keep data in sync, enhance productivity, and drive revenue.
